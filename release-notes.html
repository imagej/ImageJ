    • Thanks to MartinH, fixed a bug that caused montages created using Image>Stacks>Make Montage to lose their pixel dimensions.
    • Thanks to Norbert Vischer, fixed a bug that could cause the Pixel Inspector tool to not work correctly with blank 32-bit images.
